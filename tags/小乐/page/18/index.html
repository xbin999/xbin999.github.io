碰碰车



赶在台风来之前，去了趟儿童乐园。即春节之后，乐乐再次发挥了她在玩上面的强项。

海盗船是她的保留节目，这次排在第一位，我们去的早，还是专场，哈哈。不过我肯定是不再上去的了。



玩了些新花样，蝴蝶机，还有那个什么船“腾”地飞出去的，忘了叫什么，旋转木马，电影里常见的那种，不过都是一男一女居多。





对了，最来劲的是“穿梭时空”，转大圈圈呀，脑袋朝下，我在下面都看晕了。乐乐直转了8圈，停下来后，和她姨妈说，再反向来几圈吧。我倒！

不过好玩的在后头呀，终于见到了传说中的碰碰车呀，我们是全部上阵，并且赶紧邀请旁边的小年轻，人多热闹呀。好是一阵乱战。说来好笑呀，女儿和老爸都是第一次玩，沾了光了。

让她总结一下什么最好玩？碰碰车-穿梭时空-海盗船，排名分先后。恩，看来我只能陪她玩碰碰车了。

下一次还能玩什么呢？似乎只有过山车能过她的瘾了。

言传身教



上周特悲惨，发烧、感冒、流鼻涕，在家基本就呆了一周，套用现在流行的话，也是四十月一遇了。

而且实在对不住的是，这次的病从我开始发源，在家中都蔓延开了，全家老小尽被传上，无一幸免呀。害得我这两天天天检讨"对不住呀，对不住呀。"整一祥林嫂。

乐乐和我一样，俩人一起发烧、咳嗽、流鼻涕。

小家伙一直不太会擤鼻涕，现在既然赶上了，正好以身示范，哈哈。

"一只手按牢鼻子，嘴巴闭上，使劲，哼…"

"我看看，我看看。"

"多吧"

"哎-咦-"哈哈。

乐在儿童乐园



春节里智慧树栏目每天都是欢乐运动会，在香港迪斯尼乐园，金龟子和毛毛虫玩的不亦乐乎。乐乐说，她也要去迪斯尼乐园。

恩，迪斯尼还有点遥远，就先带她去一趟杭州的儿童乐园吧。

一开始担心小家伙是否能承受的了，先找个小儿科的。



我也上去顺便沾个光，补补小时候遗留的空白。

确实是没什么，来个刺激点的吧。



摇摆旋转伞如何，没人能陪在边上。工作人员一开始说孩子太小了吧。这家伙照玩不误啊。

接下来的这个就更刺激了，海盗船。



船摆的老高，那种失重的感觉真是太难受了。我也上了贼船，哇哇直叫。叫得最响的就是我了。乐乐却说："再高点，再高点。"

在上面拍的照片也许看不出什么，不过看看下来之后我的脸色就知道了。直接后果是后面的游戏我一个也没再玩。



青蛙跳，一群小家伙。没一个慌的，呵呵。



这个感觉象冲浪，我也忘记叫什么了。哇哇直叫，在下面看是很刺激的。



看看乐乐玩了以后是怎么说的，很过瘾吧。

[youtube=http://www.youtube.com/watch?v=kzEOKeO4VmI]

在水上钻进大球，这个应该是很爽的，就是贵了点。



蝙蝠飞车，很刺激呀。似乎是乐园里新的项目。小家伙一定要一个人坐，还不要大人陪呀。



看看这个轨道。



综上所述，小家伙凡是有的玩就没有怕的，而且还一定要刺激的。我是怕了。唉，以后去迪斯尼可怎么办呀。我的天！
