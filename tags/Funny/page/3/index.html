Gtalk talks to MSN



Gtalk其实一直使用的不多，主要还是公司里大多都是msn用户。msn比较花哨，安全就不提了。

折腾了一下，在gtalk中把所有msn的用户都导了进来，方法嘛google一下吧，主要是用了个psi软件。

这样这两天都用的是gtalk了，用了才会慢慢发现。

首先是对用户群分组，习惯了msn中的分组，虽然gtalk中地快速搜索很管用。不过很多家伙的名字都是乱七八糟的，一下子还找不着。不适应。再找找方法吧，原来这么简单。

msn的各种表情和图示没有了，不过本来就是要简单化嘛。所以大家尽量不要给我花的吧。

msn就被关掉了，也不得不关掉了，由于msn只限制用户在一个地方登陆，所以即使我再连上msn，可过不了多久gtalk登上去了，msn就又被注销了。这个还不习惯，所以干脆就关掉msn了，还省资源。先习惯着吧。

有点担心，通过服务器中转，我用的是msn.jabbernet.dk。用户名刚导入时，显示的类似xxx%hotmail.com@msn.jabbernet.dk。这样万一msn.jabbernet.dk不行了，换个服务器又要重新导一遍msn的用户呀。要真这样的话，删除原来的用户和再分组改名的不是巨烦呀。

先试着吧。

我在内部做了一下测试，收到的回复是：

36
22
0
8

最接近的是10

样本数稍微少了点，但也挺有趣。

聪明的人不少，尤其是天才型的也多。

搅局的，象白开心那样的没有。
