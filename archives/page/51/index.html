陪小乐玩24点

24点是很古老的游戏了，4张牌，加减乘除得到24，估计小时候大家都玩过。反正我小的时候是很喜欢，貌似速度也挺快的，经常在村里和小叔两人PK，往往牌还每处就喊好了。长大的时候碰到个5、5、5、1，或者7、7、3、3的题目还能小得意一番。

小乐在数字上的兴趣没我那么强，相比小说和文字对她的吸引力来说更是了，所以LP常说没得我的遗传，担心她的数学。不过也没那么在意，慢慢来呗。我就想着还是和她来玩吧，在玩的过程中能接受到一点就是一点了。
最早冒出24点的想法是很早了，至少两年前了，小家伙也是上幼儿园，加减乘除的太难了。（现在二年级也都还没算乘法呢，何必急啊！）
从小的时候我们一开始都是从认车牌开始认数字的，经常走在路上就一块块车牌点过去。于是我就想到还是在车牌上做文章，那时候还基本是在做10以内的加减法。就拿车上的数字来做运算，算12点，刚好可以突破10，有进位。于是每次一起走路，路过一辆车时，我就报数字，小乐做，顺便挑挑车，这俩BMW，那辆奔驰，呵呵。
而且难度上也不同，往往我会报4个数字，只要能用其中的任意几个数字算出12来就可以了；
再之后加了难度，每次报4个数字，要求4个数字全部都得用上；
上半年不知什么时候，可能是学校里说了一下乘法吧，忽然偶尔会来几下乘法。我于是就想是否在算12点时候也加入乘法，也没特意去背乘法口诀表（估计最熟的就是不管三七二十一和九九八十一难了），就把常用算12的几个说了说，34,26，再按加法做了验证。于是就开始允许用乘法来算12点了。也挺好玩的。
走路的时候小家伙有时会自己提出来，有时我问她，她觉得厌倦了，那就又不算，算不出来也没什么，休闲游戏。
上周路上，我又突然和小乐说，我们来加大难度吧，算24点。哈哈，她又觉得有点刺激和新鲜感。于是我们进入了一个新的里程，终于到24点了。有了38,46,2*16，也多了其他的16+8，18+6，带来了更多的乘法。
不过到现在都还不算是完全的24点，我们还没用到除法，偶尔在有些题目只能用除法才能计算的时候会顺带提一下，小乐也没特别的兴趣，那就先暂时搁着吧。等到哪一天用得上再说。

西安旅游的应用篇

在小乐暑假快要结束的时候，全家人一起去了一次西安，也就有了小乐长安行。之前在微博上曾提到了西安旅行一个很大的感受就是移动互联生活的便捷，给这次旅游带来了很大的方便性，旅行质量也提高了好多。甚至LP大人在此期间都不那么烦我一天到晚拿个手机，哈哈。

整理一下做个推荐：

  1. 大众点评 出门第一担心的就是吃饭的事情，吃什么呢？贵不贵？好不好？远不远？有没有西安特色？现在全部都不用担心。我在西安的正餐基本上都是大众点评推荐解决的，每次快要逛完景点了，就开始搜索附近500米、西安风味的餐馆，然后只要根据点评进行挑选就可以了。选择餐馆之后，再直接获取路线，就可以在地图上得到信息，左转，直行，感觉自己就是个本地人啊。在小乐面前别提有多神气了。
  2. 百度地图 其实iphone上本身的地图已经很好了，尤其是和其他应用的结合。不过出行之前我还是多选了个百度地图做后备。一来是担心Google地图在我国的特殊性，二来就是百度地图提供了离线地图的方式，我在出行前把西安和咸阳的地图都下载了。这样既省了流量，也不会因网络信号差而显示不了地图。
  3. 旅游攻略 出门前挑了挺多有关旅游的APP的，什么全国旅游景点、无线西安、陕西旅游，发现大多不靠谱，没啥实质内容。直到下了旅游攻略中，我就看了西安的，从概述、文化、交通、小吃、沟通无、旅游路线、景点介绍一应俱全，靠谱！充分体现了“内容为王”啊！另外制作也比较精美，推荐！不过我也给它提了个意见：特讨厌每次进旅游攻略，它就要提示来一下是否要联网更新旅游攻略列表？然后下面两按钮就是“客观，我不要！”，“我要！”搞得跟什么官人我要似的，特烦人。（BTW，貌似我们的APPSTORE也是类似问题，总想不断地提示别人来更新，可要是内容不行，有人来吗？）
  4. 随手记 金蝶出品，还可以同步到服务端，原先是想用来日常做记账的，不过记了没几次，就懒了。没想到这次出去旅游倒是发挥了作用，可以随手记账，所有旅行费用支出一览无遗，不用担心哪里钱数目不对了。一到家，整个这次的旅行消费就全出来了。同时还提供简单统计图表，如图：貌似以后还可以进一步先制定各块的预算，严格控制，哈哈。
  5. 其他的APP虽然下了不少，但真正起到作用的倒也还不多。比如锦江之星只能提供查询，海南航空虽然可以办理业务，但却没有把我在网站上办理的信息给同步过来，我实在是没想通。航班、公交、铁路基本都没用上，百度地图中也提供了公交线路的查询，差不多也够用了。
  6. 在网络上搞定的事情，有些虽然不是用APP，但有浏览器就能搞定很多事情了。比如航班的自助登机，可以提前选号我们一家三口的座位，就不用担心分开了。
    对了，还有必须说明的应用就是新浪微博了（我用好多个客户端，下次找个时间比较一下），旅游中多了个小乐长安行的随时播报，这也是以往从来不曾有过的。一般我都是在坐车的时候，记账、发微博，哈哈。

遗憾嘛，之前也说了，网络是基础，还是不够给力！

小乐讲故事——十二生肖演唱会

动物界最好的学校是猛猛动物学校，这是个专教勇猛的肉食动物的学校。它们的校长是大象校长。它们要开一个音乐会，还请了温柔学校的动物一起来参加。

音乐会开始了。

第一个上场的是老鼠，尖尖的声音，声音太刺耳了，狮子评委和梅花鹿就让猫把它赶下去了。

牛摇着赶苍蝇的尾巴上台了。大家嫌它的尾巴晃来晃去，有点臭，就让一个动物拿着扇子拍着它的屁股下了台。

勇猛的老虎，它以为自己是狮子主持人的朋友，所以非常的骄傲，昂首挺胸地上了台。不过看来老虎今天没睡醒，上台时打了一个哈欠，然后就开始“吼吼吼”地唱歌了。但是动物朋友们的耳朵都快被它震聋了，所以只好请它下台了。

接下来表演的是温柔的小兔子，只见小兔子歪着耳朵上来了。它的声音非常优美，把动物朋友们个个唱的想打瞌睡。兔子朋友表演完了之后，台下响起了热烈的掌声。

在小兔子表演完之后，海里的大王——龙来了，只见龙过来的时候，长颈鹿和梅花鹿给它抬着一个洗澡盆过来。梅花鹿在洗澡盆里倒满了凉水，龙在里面一边扭动着身子一边唱歌。龙唱：“奥，我是海底的大王”，由于他的声音实在是太响了，太刺耳了，所以趁它不注意，梅花鹿、长颈鹿还有观众们把它的澡盆子抬走了。

接下来是扭着小屁股的蛇上来了，这个蛇呀它在舞台上扭动着身子还一边唱着刺耳的歌。它的声音又尖又刺耳，台下的观众耳朵都快被它刺耳的声音弄聋了。为了不让观众朋友们参加一场音乐会就把耳朵弄聋了，蛇还是依依不舍的下台了。

一匹白色的马上来了，只见马跺着它的蹄子，一边唱歌，一边甩甩它的蹄子，由于它的蹄子实在是太重了，整个舞台都快被它踏翻了，大家只好拿马鞭把马拍下台了。

接下来表演是温柔的小绵羊， 只见小绵羊“咩咩”地唱着柔柔的歌，在它唱完之后，台下响起了一片掌声，小绵羊鞠了一个躬，然后腼腆地下台了。

顽皮的猴子赶紧跑了上来，只见它上来的时候，在台上弄了一盆土，种了一棵树，然后猴子爬到树上，抱着树说“我爱吃香蕉，我爱吃桃子，还有我爱吃的水果都很多。”大家因为猴子实在是太顽皮了，还把自己的尾巴乱甩，差点把小兔子观众都给卷上台去了。于是大伙儿把猴子用绳子捆了下来，送下了台。

接下来，骄傲的大公鸡上台了，只见它上来的时候在舞台上放了一堆烂稻草，然后它飞上了烂稻草，站在烂稻草上“喔喔喔”。可是烂稻草散发的臭味太臭了，公鸡也只好下了台。

再之后上台的是“汪汪汪”叫的狗。狗过来的时候带了一个狗食盆，它在里面装满了燕麦，还倒满了牛奶，在里面“呱叽呱叽呱叽”捣着。一边唱歌“我爱吃燕麦，我爱喝牛奶”，在那里“汪汪汪”乱叫，由于它吃的实在是太脏了，把舞台都弄脏了，小兔子的鼻子上还被贴了一块燕麦，它们也只好让狗狗下场了。

哎呀呀，最后一位选手肉肉猪上来了，猪上来的时候顺便放了一个屁。它带的是一个小马桶，一上来就在里面拉了一泡便便，然后唱着歌：“我猪呀，真爱拉便便，身体最最好。”由于实在是太臭了，大伙也把它赶下了台。

好，经过这次动物界的音乐大会，两位分数最高的选手被选出来了，分别是温柔的小兔子和腼腆的小绵羊。又过了三四天，森林里为它们举行了一场俩人决赛，其它选手还不错的就是猴子了。于是在那场比赛上，先给猴子挂上了一块铜牌，不过叮嘱它以后尾巴和手要管住些。

在总决赛上，请来了很多的高级老师当评委：有猛猛动物学校的总校长大象教授、温柔动物学校的语文老师梅花鹿教授，其他还有很多学校的校长和教授。

总决赛开始了，这时候，猴子同学又开始手痒了，痒什么呢？它想上去帮助小绵羊一把。结果，它推着小绵羊，把小兔子给撞飞了。等小兔子醒了之后，就让小猴子同学的爸爸妈妈把小猴子关进了自己的小房间。

总决赛重新开始进行，这次小兔子换了一首歌。这首歌的名字叫《美丽的森林》，而小绵羊唱的歌是《美丽的草原》。小兔子的声音比原来更好听了，它还穿着一件漂亮的公主裙呢！接下来，小绵羊上来了，小绵羊“咩咩”地唱着“草原真美丽”。

唱完了之后，评委们评分，结果是小绵羊获得了胜利。不过它们两个之间只差了半分，这是为什么呢？小兔子怎么会和小绵羊只差了半分呢？因为呀，小兔子在上台的时候无意间把裙子给钩破了，被评委看到了，扣了半分，最后小绵羊拿到了一枚金牌。最后的奖品是：小绵羊了获得了一顶遮阳帽，送给了小兔子是一个漂亮的蝴蝶结，至于小猴子嘛，得到了一串香蕉。

音乐大赛就此谢幕。不过其它动物也都有一份奖励，每人得到了它们一份爱吃的东西。

好了，故事讲完了。

 

——这是小乐拿着家里的实用气象历书（见上图，估计现在的小朋友家里都不太能见到了）讲的一个故事，我做了录音，然后整理得到的，改了一些口误和衔接的文字。

这个站点是怎么建起来的

心血来潮想自己建个站点，不过没想花多少钱，所以能免费的就先免费着。

  1. 之前曾看到过采用Amazon Web Servcie有免费一年的优惠，自主性也比较强。所以又找了一下那几篇文章《免費玩雲端運算，Amazon Web Service 雲端運算平台攻略 》，有1，2，3系列，就依样画葫芦来了一遍。AWS帐号原先注册过，申请了EC2；
  2. 确认是免费后，再架了个wordpress。上面的那篇教程写的非常详细，基本没遇到什么问题；
  3. 接着就是申请域名了，看介绍都说选择国外的域名注册好，那我也就随大流了。选择了GoDaddy，支持支付宝付款，好像是7美元多吧。又发现有好的教程，好心人真是多；
  4. 简单设置一下，将IP和域名做个绑定，可以访问现在这个博客，取了个名字叫杨先生村民，也给俺老家做个宣传；
  5. 余下的就慢慢捣鼓吧，我也不熟悉。从原先的wordpress（老被墙）到了最近的几篇文章，免得都是空白；再就是装了几个wordpress的plugin，先凑乎着吧；

陪小乐做聪明题

上周末的时候小乐的聪明题里有这样一道题目：

数学-学数=36，然后要填：数=( )，学=( )

从大人的思维来解这道题目：

列方程：10x+y-(10y+x)=36

化简后得：9(x-y)=36，也就是x-y=4

所以答案应该是有多组解：(5,1),(6,2),(7,3)…

不过怎么和小乐说呢？

先还是从凑数字开始吧，我们一起凑了：

数=5，6，7

学=1，2，3

“有什么规律呢？”

答案不言而喻，小乐说：上面是5，6，7，依次往下数，下面是1，2，3，依次往下数。

所以我们就又很容易得到了84，95。

“还有什么规律吗？”

4-1，5-2，6-3，两个数相减都为4。

不过这时候，小乐又凑出的一个数字就让我吃惊了一下，因为她给出了40-04，这一下子就突破了我的惯性思维，总以为二位数是是没有0的。这使得题目也好解了很多，变成了两位数减一位数。也使得我重新再出一道题目的时候，可以很容易从0开始算起。

比如：数学-学数=27

小乐又从0开始推，很简单就得到了30-03=27，然后之后就是十位数，个位数往上加1了。使得我讲题目一下子轻松了好多，这个0实在是太秒了，击掌祝贺一下！

那么是不是题目就到此结束了呢？

其实还不然，这时候，我让小乐自己来出一道题目看看，出一道题目给她老妈做。这时候可就又有点难了！下面的差也不是随便能凑的，呵呵。

于是，我们就一起继续找下面差的规律，36，27，18…

小乐告诉我的第一个规律，又让我吃惊了一下，她说两个数字相减是3，5，7，都是奇数。哈哈，果然是，这又是我没想到的。

“我们再看看两个数相加会怎样？”

3+6，1+7，1+8，都是9，所以如果再出一个题目的话可以是45，54。

这里的答案其实在上面的式子中9(x-y)=36，要求减数必须是9的倍数。不过这个等以后再和小乐再说了。

数学还是很有趣的，陪小乐的时间太少了点，不然一定能一起找更多有趣的规律来。

我的iphone4 越狱

原先的E71一直接听效果都不太好，在收到招行的广告邮件后，忍不住就下了单，入了一个iphone4 32G，（当初只看到有32G的【聚便宜】，现在倒是有16G的了）忍了几天就想越狱了。

不过之前还是做了点准备工作，害怕白苹果的说：

说说两次越狱的经验吧。

第一次：

  1. 机子是行货，os 4.2.1，用TinyUmbrella备份shsh，可惜已经无法提供4.2.1的备份了。当时4.3.1的完美越狱也还没有出现，所以貌似我也只有成功一条路，否则就只能恢复到4.3.1的了。
  2. 不过当时也还没怎么明白，胆子大就直接开工了。当时阿毛刚和我说他越狱失败，恢复到4.3.1了，呵呵。
  3. 采用的是绿毒，进入DFU模式的那几下按键真是觉得复杂，真搞不明白干嘛要这么折腾人。第一次我就没进入DFU模式，只有再把关机、开机、开机+home、home重来一遍，还好接下来倒是顺利。
  4. 再开机的时候出现了骷髅头，一阵惊喜啊，进去后也看到了cydia的图标，一块石头落地，觉得越狱也不过如此。（注意在设置中把锁屏时间设置为从不，否则1分钟就锁住了）
  5. 意想不到的事情接着就出现了，cydia做初始化的时候，我突然担心它要联网（而公司的WIFI是需要验证的，我起来的时候没有在浏览器上做验证），于是没多想就按Home退了出来。（其实是我多想了）
  6. 之后就悲剧了cydia进不去了，再重启，cydia的图标不见了，另外很多系统的图标都不见了，ipod，通讯录等等。
  7. 我只有再越了一次，cydia重新出现，越狱成功，可惜系统图标还是不见。
  8. 我不想恢复到4.3.1，不然就无法完美越狱，只有先如此将就着，好在既然越狱了，还是能找到很多替代软件的，感觉也还行。
  9. 后来曾想解决一下这个系统图标丢失的问题，看了com.apple.mobile.installation.plist文件，是一个xml的系统已安装应用的配置文件，不过删除了还是没起到作用。另外我也看到在我iphone系统中有些系统应用连目录也不存在了。
  10. 教训就是在cydia启动初始化的时候千万别退出。

第二次：

  1. 不完美了几天，没想到4.3.1的完美越狱很快就推了出来，这下子我就又来精神了。
  2. 第二次开始折腾算是有了点经验，这次是采用红雪，看教程还是挺简单，另外ray告诉我他已经越了，就更无需担心了。
  3. 首先是先将我的机子恢复到4.3.1，这个本来就是apple官方支持的，但偏偏我就在这里载了跟斗。恢复过程中还顺利，但最后验证时却出现了1004错误。
  4. 于是机子小白了，折腾啊，google，发现是hosts文件的问题，就把“gs.apple.com”的解析删掉，重新来还是不行，来来回回折腾了n回，我要绝望了，只有重启机器，再把hosts文件的解析去掉，重新恢复发现ok过关。后来想想当时折腾的时候都没有关闭itunes，所以就是改了hosts文件也没有其作用，而只是在cmd窗口中ping了一下做验证。而阿毛当初顺利恢复，是因为他根本没有备份shsh，所以hosts中“gs.apple.com”也没增加指向到cydia的服务器。不过我这样也有点担心即使备份了4.3.1，那以后APPLE不提供验证了，要连接cydia的服务器验证还是挺麻烦的一件事。
  5. 之后的过程就顺利了，有了上次的经验，进入DFU模式就容易了，顺利越了狱；

不实践总是很难得到经验的，两次过程虽然都有点心惊胆战，但解决问题后的喜悦就不言而喻了。好久没写blog了，这次倒是写长了点，以后找时间再来推荐几个工具吧。
