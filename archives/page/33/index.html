童心

每年高考作文结束后，都照例会有一堆的讨论，范文，不管是高分的还是搞笑的，以及对各高考题目的评头论足。

钱江晚报9日的“众议高考”有两篇评论：
一篇是“失去孩提王国，教育也将沉沦”，举到一个例子，教学是标准化的，题目要求和“照耀”组词，标准答案是“阳光”，答了“太阳”就是错。作文在一定程度上也是标准化的，而标准化的第一条原则就是“主题思想正确”，或“中心观点正确”，而不是感情的真实、观点的真实。
想到小乐的那个有趣的夏天的故事，最后的故事结尾是西瓜被老师给瓜分了，看来是属于思想不正确了。

一篇评论是“教育与学生一样渴望解放童心”，其中说到“高考作文的假大空是果，又是因，教师与学生同时背这恶性循环裹挟着”。小学生作文竟然无话可说，有人归咎于“教师指导作文不力”，其实恰恰相反，是“太用力”了，早早让小学生作文内容戴上了面具，结构戴上镣铐，让他们背负内容与形式的庞大教条，乃至于人生说谎作文始，人生老成作文始，怎么可能让他们活泼的童心、纯真的情感保鲜。
小乐的作文在写着，写着的时候，我常说她有点虎头蛇尾，开篇往往比较宏伟，但到最后不知是急于要收尾呢，还是要套入”啊XXXX，多么XXX，我好XXX”的抒发感叹系列中，这实际也是结构的标准化。

其实学校与老师们也深刻地懂的这个道理，但他们也受制于这个教育体系，考核学校、考核老师的KPI指标是什么呢，还是考试的分数，所以最终走向了标准化，而忽视了童心的宝贵。

给小乐讲了个美国幼儿园起诉案的故事，是在朋友圈中看到的，说的是美国幼儿园的一位母亲因为女儿认识礼品盒上的一个字母”O”，而把幼儿园告上了法庭。因为幼儿园教孩子认识”O”的同时，剥夺了孩子的想象力，而孩子本来会把”O”说成苹果、太阳、足球等等圆形的东西，现在孩子的这种能力失去了。最后这位母亲赢得了这场官司，她在辩护中又讲了一个故事：
“我曾到东方某个国家旅行，在一家公园里见过两只天鹅，一只被剪去了左边的翅膀，一只完好无损。剪去翅膀的被放养在较大的一片水塘里，完好的一只被放养在一片较小的水塘里。当时我非常不解，就请教那里的管理人员。他们说，这样能防止它们逃跑。我问为什么?他们解释，剪去一边翅膀的无法保持身体平衡，飞起后就会掉下来；在小水塘里，虽然没被剪去翅膀，但起飞时会因没有必要的滑翔路程，而老实地呆在水里。当时我非常震惊,震惊于东方人的聪明。可是我也感到非常悲哀，为两只天鹅感到悲哀。今天，我为我女儿的事来打这场官司，是因为我感到伊迪丝变成了劳拉三世幼儿园的一只天鹅。他们剪掉了伊迪丝的一只翅膀，一只幻想的翅膀，人们早早地就把她投进了那片小水塘，那片只有ABC的小水塘。”

对这个故事的真实性，我有一些怀疑，不过和小乐讲这个故事，还是想和小家伙说童心的可贵，不必追求所谓的对错，同样真实也远胜于华美。

钱报的评论说：这样看来，今年浙江的高考作文题出得真好：“世界正在失去伟大的孩提王国。一旦失去这一王国，那是真正的沉沦。”

小乐听完故事后，诗兴大发，写了首：

童心在哪里

童心，在孩子的言语里。

一声亲切的问候，

一声快乐的欢笑，

都包含着属于孩子的童心。

童心，在孩子的行为里。

一个善意的恶作剧，

一张有趣的贺卡，

都隐藏着强大童心。

童心，还在许多地方，

其实大人也能拥有童心，

和孩子一样的童心，

只是你要把童心找回来，找回来…

时间阶梯

历史的阶梯是比这幅图的底部还要更遥远、更遥远的过去开始，一直这样向上、向上、向上，直到我们现在的位置——每一个台阶都是一百年，每一层楼梯是一千年。这个楼梯可以一直不断地向前升，直到高空。让我们站在现在的位置，回顾下面的楼层，倾听那些发生在过去漫长岁月中的故事。
这个就是时间楼梯，希利尔在《希利尔讲世界史》中专门介绍了这样一个阶梯形状的图标能使孩子对于世界历史的时间范围和发展阶段有一个形象的认识。

看到这个图的时候，我就觉得这玩意真不错，画一个在经常看见的地方，能直观地感受到重大事件在历史长河中的位置，并且也不会觉得那么孤立。于是趁着假期，就鼓动小乐一起动手，参照书中的附图，画了一个“时间阶梯”图，从石器时代一直到公元2000年，一层是1000年，一个台阶100年，画的时候感觉时间好短暂，我就这么轻轻一横100年就那么过去了，呵呵。在每个100年的事件标注中除了参照原图中的重大事件，抹去了一些我们不熟悉的内容，同时最重要的是我们加入了中国历史朝代。

不知道大家在学习历史的过程中有没有这样的感觉，就是中国历史和世界历史是脱节的，以前学历史的时候也总是背843年查理曼帝国一分为三，1789年7月14日法国大革命，但从来没搞清当时的中国是在干嘛？所在在和小乐画时间阶梯的时候，我们把同一朝代在阶梯上用相同颜色的线段加粗，同时标注朝代的起止时间，这样我们就看到了希腊举行第一次奥林匹克运动会的时候是在西周，十字军东征是在南宋的时候，有趣吧！

希利尔建议有必要让孩子们把“时间楼梯”牢牢记住，就像“九九乘法表”一样，经常记忆，知道能够详细说出与每个日期相关联的时间。这样做的目的是让孩子们能够对从古至今的世界历史做出一个大致的概括，并可以说出重要的历史日期和历史事件，不会由犹豫，不犯错误。听起来有点难，但值得尝试一下，我再想想办法看，大家有什么好方法也能提提建议。

Ruby中正则表达式和输入输出



使用/pattern/创建一个正则表达式，正则表达式的详细语法就不说了，无外乎：* 0到多个字符, + 1到多个字符, | 或,() , \d 数字, \s 空格, \w 字母, . 任意字符。

=~ 操作符用于字符串匹配正则表达式，匹配成功返回字符串中匹配的位置，不成功则返回nil。

sub用于单个替换，gsub用于单个替换。

1.9.3p125 :095 > “ttPerl” =~ /xx|Perl|Python/
=> 2
1.9.3p125 :096 > “ttPerl”.sub(/Perl/, ‘Ruby’)
=> “ttRuby”

给小乐讲“希利尔讲世界史”之十字军东征

最先开始小乐选了一篇“没人喜欢的约翰”，众人爱戴的狮心王理查德有一个兄弟名叫约翰，和理查德正好相反，没人喜欢约翰。

“狮心王理查德怎么是受众人爱戴的呢？”小乐有点疑惑，她可能在其他书上看到过。

不过希利尔所写的理查德是个英雄，让我们跳到前面的章节“画连线——三个国王成一行”。

这里有三个国王：英国的理查德，法国的菲利普，德国的腓特烈.巴巴罗萨。上个故事里讲到耶路撒冷被基督徒夺回来了，但好景不长，穆斯林发起进攻，又把它夺了回来。

 

看来跳着念故事还有点难度，希利尔老是给你上文说到，我跟小乐说：“要不爸爸再先给你说说前面的一次伟大的历险。”

“没关系的，继续说好了。”

 

好吧，基督徒又发起了第二次十字军东征。从那以后，在接下来的两百年时间里，他们发起了一次又一次的十字军东征，共有八九次。

 

这三个国王参加的是第三次十字军东征，也就是将近公元1200年的时候。德国的腓特烈.巴巴罗萨。腓特烈是他的姓，他的名字是巴巴罗萨。

“好奇怪，怎么是姓在前面的。”

巴巴罗萨这个词的意思是“红胡子”。其实这是国王的绰号，当时给国王起外号可是个风俗哦！当时腓特烈国王的都城在亚琛，查理曼大帝也曾在此定都。不过腓特烈与查理曼大帝不同，查理曼是包括很多国家在内的庞大帝国的国王，而腓特烈只是德国的国王。当三个国王联合起来发动第三次十字军东征的时候，他的年纪已经很大了，他最后没有到达耶路撒冷，因为半路过河的时候他掉进河里淹死了。腓特烈的情况就是这样。

第二个国王是法国的菲利普。他非常嫉妒第一个国王，也就是英国的理查德，因为理查德非常受十字军的爱戴，结果菲利普中途放弃东征，回法国去了。

理查德为人温和仁慈，又坚强勇敢，人们都叫他“狮心王理查德”。他对犯错的人惩罚非常严厉，但却能做到大公无私。人们都很爱戴他，同时也对他充满敬畏，因为他嫉恶如仇，对坏人毫不留情。

理查德后来和耶路撒冷的穆斯林国王萨拉丁成了朋友，达成了友好约定，萨拉丁答应以后不再对耶稣墓和来朝圣的信徒不敬。

 

理查德还有很多有趣的故事，不过今天主要说说一个更好玩的。是发生在第四次十字军东征后，在1212年又组织过一次，这次的十字军战士都是些小孩子，所有又叫儿童十字军，或叫童子军。这只童子军是由一个12岁的法国男孩率领的，他的名字叫司提反，和第一位基督教的殉难者同名。

我们怎么都想不明白，他们的爸爸妈妈怎么会同意的呢？

“是呀！”小乐也感到十分的困惑。

童子军一路向南部的地中海行进，他们以为到底这里后，海水就是自动向两边分开，然后中间会出现一条大路直通耶路撒冷，就像在《圣经》里说以色列人离开埃及的时候，红海的水自动分开给他们让路。但现实中的海水并没有给他们让路。

“那他们怎么办呀？”

这时，他们遇到了一些水手，这些水手提出主动要用船把童子军送到耶路撒冷去，他们还说自己不要报酬，这样做完全是对上帝的爱。

 

小乐赞：“他们真不错。”

孩子们上了船后才发现这些水手实际都是海盗，他们直接把船开到了十字军的敌人——穆斯林的地盘，并把孩子卖给了穆斯林做奴隶。如果是在《格林童话》中，也许孩子们还能设圈套把海盗们给抓起来，可现实是很残酷的，所以，我也没法让这个故事有个完美的结局，因为事实原本如此。

希利尔这段讲得真是非常有趣，对孩子来说也看到原来历史不是故事，有它自己残酷性。

 

不过对这段童子军的历史，还是觉得很困惑，又查了一下维基百科，上面说到：

第四次十字军东征结束8年后（1212年），传说有30,000名儿童组成了一支儿童十字军（Children’s Crusade，也称童子军）。不过现在学界普遍认为这支队伍实际上并不是由儿童，而是由流浪汉组成的。这些人基本上没有人成功到达圣地，其中有一些人没能回家而被贩卖为奴。

每次给小乐解惑的时候，我都会感叹维基百科带来的帮助，谢谢维基百科。

Ruby中变量、数组和哈希



Ruby中有关命名的习惯，使用第一个字符来区分名字的用途：

1. 局部变量、方法变量和方法名称统一用小写字母或下划线开始。
2. 全局变量加一个$前缀。
3. 实例变量以@开始。
4. 类变量以@@开始。
5. 类名称、模块名称、和常量采用大写字母开头。
6. @后面不能直接跟数字。

数组（Array）和哈希（Hash）都是索引集合，区别在于数组的下标是整型，而哈希可以用任意对象作为key值。

数组和哈希存储的对象可以是任意对象，而且可以不同。 使用方括号[]存取。
初始化方式，可以采用%w做初始化来简化引号和逗号的书写。

> a = [1, ‘cat’, 3.14]

> empty2 = Array.new
> b = %w{ ted xbin mary}

Ruby中单引号和双引号的区别



Ruby中使用引号括起字符表示字符串，单引号和双引号有所不同，单引号所括的内容不会做转义，直接就是字符串的内容；而在双引号中，Ruby会做更多事情。首先Ruby会查找转义符‘\’，并做相应的替换，典型的如换行’\n’；其次是表达式求值，对于字符串中的#{expression}会被表达式的值所替换。

1.9.3p125 :060 > puts “Hello\nworld”
Hello
world
=> nil
1.9.3p125 :061 > puts ‘Hello\nworld’
Hello\nworld
=> nil
1.9.3p125 :062 > name = “Ted”
=> “Ted”
1.9.3p125 :063 > puts “Hello,#{name}”
Hello,Ted
=> nil
1.9.3p125 :064 > puts ‘Hello,#{name}’
Hello,#{name}
=> nil

惩罚和承受的不同

转乐妈的文章：

晚上带乐乐去上兴趣课，停着等红灯。看到人行道上，一位30出头的爸爸，带着2岁多的儿子，在玩拍气球的游戏。气球总是一晃一悠、起起伏伏，在空中逗着父子俩。一阵风吹来，气球被男孩一拍，一路晃呀晃的，跑到了人行道上，爸爸三步并作两步还是没有追上。好不容易抓住了气球，小儿子开心的迎上去，伸出双手，昂着头，想从爸爸手中接过它。可是只听“呯”一声，爸爸一手捏爆了气球，：“谁让你拍到马路上去的！” 男孩子被突如其来的变故先是惊呆了，而后看着撒落在地上的碎片，就大哭起来。

这个爸爸对孩子做了他认为正确的惩罚。我问乐乐：“这个爸爸对吗？” “不对，对他儿子这个年龄，这是很正常的事！希望他的妈妈能慈爱一点！” 是的，乐乐一下就说到了点子上，在当爸妈之前，真应该好好学学孩子的心理学。什么年龄的孩子懂什么，不懂什么，我深有感触，自己也一路犯了不少这样的错误。作为家长确实有时很难免，但就算真是孩子错了，除了惩罚还有别的方法可以选择吗？

上周五，乐乐的作业其实不多，按她的速度2小时可以搞定，但她硬是用了3小时。我在外面坐着，虽然手中捧着书，但却是无心看下去，想着怎么和孩子沟通。以我的功力，我其实是很强迫很为难自己的：“不要动怒，好好说，不要冲动，想想书” 脑子里不停地劝自己。现在的我很少会想到惩罚孩子，真心想和孩子一起解决问题，只是实际操作时还是会很僵硬，不像LG那么自然。还属于嘴上不说，但心里还是没有谱的阶段。

终于，乐乐出了房间，让她面对面坐在我的腿上，”乐乐，妈妈多想能晚上带你出去玩，也多想能有时间给你看书，可是今天这样，就没有办法出去玩和看书了，今天在做作业上，是什么原因让你晚了近一个小时呀？你有什么办法能提高效率吗？“ 乐乐双眼闪烁不定，轻咬嘴唇，“恩，做作业前先上好厕所，喝好水，然后把桌上玩的东西拿开！” 我觉得这次自己终于忍住了，没有直接给孩子建议，让她自己去想去思考。而且她提的方法也不错，这件事算是平和解决了。

后来，看到书上有一段话：惩罚和承受自然结果有什么不同？ “惩罚是为了教训孩子，故意剥夺他们的时间或者追加他们的痛苦，承受自然后果是让孩子承受自己行为所产生的自然结果。” 我是事后才看到这段话的，我当时的口气没有惩罚的意思，告诉了她，这么做的结果是什么，让孩子自己想方法来解决它。第二天带她去外面玩，路上有一个妈妈对自己孩子很是凶，乐乐转过来抱着我：“妈妈不会这样对我了，昨天也是好好和我说的！” 我心里也很开心，自己再一次克服了自己，这次没有让冲动变魔鬼。

昨天，在书上也看到，孩子有时出于很多原因，会对自己作出的许诺无法履行。我回想自己，乐乐有时也有说到，却没有做到的事，我当时还是很动怒的，认为孩子说话不算话，其实这也是她心智没有长大造成的。我不想设陷阱去考验孩子，我想善意的提醒她，于是就拿起笔，又做了一回字条妈妈，放在她的写字台上。等她做好作业时，她开心的拉着我，“妈妈你看！” 桌上除了笔什么也没有，还递上那张字条，上面写着她特殊的语言“ga ga “!
