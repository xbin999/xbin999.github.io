给小乐讲“希利尔讲世界史”之十字军东征

最先开始小乐选了一篇“没人喜欢的约翰”，众人爱戴的狮心王理查德有一个兄弟名叫约翰，和理查德正好相反，没人喜欢约翰。

“狮心王理查德怎么是受众人爱戴的呢？”小乐有点疑惑，她可能在其他书上看到过。

不过希利尔所写的理查德是个英雄，让我们跳到前面的章节“画连线——三个国王成一行”。

这里有三个国王：英国的理查德，法国的菲利普，德国的腓特烈.巴巴罗萨。上个故事里讲到耶路撒冷被基督徒夺回来了，但好景不长，穆斯林发起进攻，又把它夺了回来。

 

看来跳着念故事还有点难度，希利尔老是给你上文说到，我跟小乐说：“要不爸爸再先给你说说前面的一次伟大的历险。”

“没关系的，继续说好了。”

 

好吧，基督徒又发起了第二次十字军东征。从那以后，在接下来的两百年时间里，他们发起了一次又一次的十字军东征，共有八九次。

 

这三个国王参加的是第三次十字军东征，也就是将近公元1200年的时候。德国的腓特烈.巴巴罗萨。腓特烈是他的姓，他的名字是巴巴罗萨。

“好奇怪，怎么是姓在前面的。”

巴巴罗萨这个词的意思是“红胡子”。其实这是国王的绰号，当时给国王起外号可是个风俗哦！当时腓特烈国王的都城在亚琛，查理曼大帝也曾在此定都。不过腓特烈与查理曼大帝不同，查理曼是包括很多国家在内的庞大帝国的国王，而腓特烈只是德国的国王。当三个国王联合起来发动第三次十字军东征的时候，他的年纪已经很大了，他最后没有到达耶路撒冷，因为半路过河的时候他掉进河里淹死了。腓特烈的情况就是这样。

第二个国王是法国的菲利普。他非常嫉妒第一个国王，也就是英国的理查德，因为理查德非常受十字军的爱戴，结果菲利普中途放弃东征，回法国去了。

理查德为人温和仁慈，又坚强勇敢，人们都叫他“狮心王理查德”。他对犯错的人惩罚非常严厉，但却能做到大公无私。人们都很爱戴他，同时也对他充满敬畏，因为他嫉恶如仇，对坏人毫不留情。

理查德后来和耶路撒冷的穆斯林国王萨拉丁成了朋友，达成了友好约定，萨拉丁答应以后不再对耶稣墓和来朝圣的信徒不敬。

 

理查德还有很多有趣的故事，不过今天主要说说一个更好玩的。是发生在第四次十字军东征后，在1212年又组织过一次，这次的十字军战士都是些小孩子，所有又叫儿童十字军，或叫童子军。这只童子军是由一个12岁的法国男孩率领的，他的名字叫司提反，和第一位基督教的殉难者同名。

我们怎么都想不明白，他们的爸爸妈妈怎么会同意的呢？

“是呀！”小乐也感到十分的困惑。

童子军一路向南部的地中海行进，他们以为到底这里后，海水就是自动向两边分开，然后中间会出现一条大路直通耶路撒冷，就像在《圣经》里说以色列人离开埃及的时候，红海的水自动分开给他们让路。但现实中的海水并没有给他们让路。

“那他们怎么办呀？”

这时，他们遇到了一些水手，这些水手提出主动要用船把童子军送到耶路撒冷去，他们还说自己不要报酬，这样做完全是对上帝的爱。

 

小乐赞：“他们真不错。”

孩子们上了船后才发现这些水手实际都是海盗，他们直接把船开到了十字军的敌人——穆斯林的地盘，并把孩子卖给了穆斯林做奴隶。如果是在《格林童话》中，也许孩子们还能设圈套把海盗们给抓起来，可现实是很残酷的，所以，我也没法让这个故事有个完美的结局，因为事实原本如此。

希利尔这段讲得真是非常有趣，对孩子来说也看到原来历史不是故事，有它自己残酷性。

 

不过对这段童子军的历史，还是觉得很困惑，又查了一下维基百科，上面说到：

第四次十字军东征结束8年后（1212年），传说有30,000名儿童组成了一支儿童十字军（Children’s Crusade，也称童子军）。不过现在学界普遍认为这支队伍实际上并不是由儿童，而是由流浪汉组成的。这些人基本上没有人成功到达圣地，其中有一些人没能回家而被贩卖为奴。

每次给小乐解惑的时候，我都会感叹维基百科带来的帮助，谢谢维基百科。

Ruby中单引号和双引号的区别



Ruby中使用引号括起字符表示字符串，单引号和双引号有所不同，单引号所括的内容不会做转义，直接就是字符串的内容；而在双引号中，Ruby会做更多事情。首先Ruby会查找转义符‘\’，并做相应的替换，典型的如换行’\n’；其次是表达式求值，对于字符串中的#{expression}会被表达式的值所替换。

1.9.3p125 :060 > puts “Hello\nworld”
Hello
world
=> nil
1.9.3p125 :061 > puts ‘Hello\nworld’
Hello\nworld
=> nil
1.9.3p125 :062 > name = “Ted”
=> “Ted”
1.9.3p125 :063 > puts “Hello,#{name}”
Hello,Ted
=> nil
1.9.3p125 :064 > puts ‘Hello,#{name}’
Hello,#{name}
=> nil

Ruby和其他面向对象语言的一点差异



Ruby是一门真正的面向对象的语言，任何操作的东西都是对象，包括操作结果本身。

song1 = Song.new(“Ruby Tuesday”)
song2 = Song.new(“Enveloped in Python”)
# and so on
</pre> 在Ruby中定义一个类(class)，通过constructor new来创建对象(object)，这些实例(instance)从同一个类衍生，但有不同的唯一标识。你可以定义不同的实例变量，持有对象的状态，比如每首歌。 每个类可以定义实例方法，以存取对象的实例变量。通过给对象发送一个消息来调用方法，消息包括方法名称，所需要的参数。当对象接收到一个消息，它会查找类本身是否能响应方法，如果能找到，执行方法。 看下面的例子： <pre>
“gin joint”.length » 9
“Rick”.index(“c”) » 2
-1942.abs » 1942
sam.play(aSong) » “duh dum, da dum de dum …”

点前面的称为接收器(receiver)，之后的是被调用的方法。

这里有一点Ruby和大多数其他语言的不同。比如求绝对值，Java会调用一个独立的函数并传递参数：

number = Math.abs(number) // Java code

但在Ruby中，求绝对值是内置在数字本身的，内部关注细节和完成处理，你只需要简单地发送一个abs消息给数值对象即可。

希利尔讲世界史之拿破仑和回文

“猜猜这个人是谁？”

“拿破仑.波拿巴。”

拿破仑拿破仑出生在地中海一个叫科西嘉的小岛上。其实以前，科西嘉还是属于意大利的，在拿破仑出生之前的几天才被划归了法国，他成为了一名法国人。等他长大，就被送到法国的一个军事院校去学习。很快，拿破仑就展现出具有很强的军事天分，能力杰出。在他26岁那年，就已经成为了一名将军。此时，法国大革命已经爆发，人民除掉了国王，摆脱了国王的统治。欧洲其他国家害怕他们的子民也染上这种革命热潮，因此十分痛恨法国人民。而法国人还去邻国，帮助他们除掉自己的国王。于是，又一次的战争不可避免的爆发了。

拿破仑也被派去了国外作战，到意大利去进攻。途中必须翻过阿尔比斯山，要知道，想翻过这座巨大的山脉可不是一件轻而易举的事情，而且拿破仑的军队还带着沉重的大炮呢。管理大炮的技师认为不可能把大炮带过山，他们在这个问题上的认识是很有权威的。可是拿破仑可不信那套，他认为只有蠢人的字典里才有“不可能”三个字。他身先士卒，带头向前冲，命令士兵们不要想这是在过山。在他的带领下，军队果然都翻过了山，并且没丢弃一个大炮。在与意大利的战争中，拿破仑的军队胜利了，回国的时候成为了凯旋的英雄，受到人民的热烈欢迎。

后来拿破仑征服了大部分欧洲的国家，如西班牙、普鲁士、奥地利等，于是他将目光瞄向了下一个国家。

“老爸，你给我点提示。”

“你想想我们玩欧洲之旅时的欧洲地图，法国和英国隔海相望，他又放弃了征服英国的想法，意大利老早就被打败了，西班牙在下面，普鲁士和奥地利相当于现在的德国，拿破仑想征服的是欧洲最大的地方。”

“俄罗斯。”

对了，俄国，而这一决定成为他一生中犯的最大错误。他没有考虑清除俄国的具体环境就贸然出击。俄国距离法国很远，虽然法军安然抵达俄国中心莫斯科，但俄国幅员辽阔，战线一拉就是很长，而且俄国人把法军的粮食销毁了，天气又十分寒冷，军队损失惨重。拿破仑的计划失败了。

拿破仑眼看大势已去，就签署了退位诏书。他被放逐到意大利一个叫厄尔巴的小岛上。拿破仑到了岛上以后，并没有灰心，他认为自己仍有机会回到法国，重掌大权。他秘密准备了很久，终于有一天，突然在法国海岸登陆了，惊动了法国甚至整个世界。巴黎的政府派人去抓捕他并把他押送到巴黎。可谁知道派去的人都是拿破仑的旧部下，遇到这位从前的将军后，都心甘情愿地站在了拿破仑的一边，跟他回了巴黎。

英德联军聚集在法国的北部，由一位叫威灵顿的将军带领，准备和拿破仑决战。拿破仑迅速组织了一支队伍前去迎战。在一个叫滑铁卢的镇上，双方激战，最终拿破仑被彻底击败了。

“是的，滑铁卢战役。”小乐说。

这场战役发生在1815年，今天我们讲到重大失败时，就会比喻为遭遇了“滑铁卢”。在滑铁卢一役之后，英国人怕拿破仑再次跑掉，就把他关在大海中的一个叫圣赫勒拿的小岛上。拿破仑在岛上生活了六年，便去世了，结束了他的戎马生涯。

这里有个句子，据说是拿破仑战败后说的：“ABLE WAS I ERE I SAW ELBA.”意思是“在回到厄尔巴岛之前，我是所向无敌的。”

不过你看出其中的奥秘了吗？这句话是英语的回文，类似中国古代的回文诗，顺着读和倒着读是一样的。

小乐仔细地看了，惊奇地说：“真的！”

“老爸给你说一个现代的回文，上海自来水来自海上。”

小乐转了转脑袋，“呀，真是的，好玩。”

“嗯，我再给念一个对联，说的是一个和尚画了一幅荷花，出了个上联，叫做‘画上荷花和尚画’。”

“好厉害，那下联呢？”

“古代有个官职称为翰林，你知道的吧？”

“嗯。”

“书法中参考别人的字帖被称为临帖，翰林模仿汉朝的字帖写字，这下联就是‘书临汉帖翰林书’。”

小乐啧啧称奇，觉得这回文太神奇了。

我看她起劲，就说：“要不我们自己来编回文吧。”

“怎么编呀？”

“正念反念都可以，字数少一点没关系。比如老爸先来一个‘大哥大’。”

“啊，大哥大是什么？”

移动电话也没多少年，现在的孩子已经都不知道这手机的老祖宗了。

“大哥大就是早期的手机，体积比较大。”我一边说一边比划，“拿在手上这么大，象个砖头一样，以前拿着的人觉得特别有派头。”

“长发长。”乐妈来了一句。

“短发短。”小乐立即来个反义词，第一个回文编出来了，就简单多了。

“不错，三个字，我们再来长一点，五个字的，乐爱爸爱乐。”

“我说一个不实际的，上海在海上。”

“这个也不算不实际，还行。”

“还有一个，奶牛挤牛奶。”

两人嘻嘻哈哈编得啥是有趣，最后来看看希利尔对拿破仑的评价吧。

拿破仑可能是历史上最伟大的将军，但他的为人却不受人们肯定。他为了成就自己，而不顾整个欧洲人的生命，随意展开战争，使人民生灵涂炭，因此，他死后并不被人们所尊敬。

给小乐讲“希利尔讲世界史”之红帽子和红高跟鞋

最近都在给小乐念“希利尔讲世界史”，小家伙挺喜欢，我又从乐妈那里抢回了每晚的“念书权”，嘿嘿。不过讲的方式和以往不同，不是按顺序往下讲，有时我会说：“想听哪个国家啊？”“美国。”这时我就会找一篇美国的故事；而有时就直接在1～90中选一个数字，根据key值直接定位故事内容，搞IT的人呀，什么都往写代码的思路靠。

今天给小乐讲“红帽子和红高跟鞋”。

要讲的两个国王是路易十三和路易十四，他们对法国的统治也是在17世纪，和斯图亚特家族统治英国是同一时期。

路易十三只是名义上的国王，一切权利都在另外一个人受理。真正掌握大权的人是教会的最高管理者，也称为红衣主教。红衣主教们都戴着红帽子，穿着红色长袍，而这位红衣主教名叫黎塞留。

在路易十三和黎塞留之后的下一任法国国王是路易十四。

当时英国的民众经过长期斗争，最终成功地把管理国家的权利交到他们的议会手中，但在法国，路易十四可不干，他声称：“我就是国家。”

“LIAN即国家”小乐说：“啊，不对，朕即国家。”

小乐看来听到过这句话，「朕即国家」这句话也有历史学家认为这种说法不准确，据报道，和此话原意相反的是，路易十四在临终时说的话是：“朕走了，但国将永存。”

路易十四在位七十多年，是历史上在位时间最久的一位国王。

小乐问：“路易十四活了多久？”

“我查查看，1638年9月5日－1715年9月1日。”

“那是77岁，那他7岁就当国王了。”

“从1643年到1715年在位，长达72年，5岁就当国王了。”

路易十四被称为“显摆国王”。

我问小乐“显摆是什么意思？”

“我知道它的意思，但我说不上来。”

“有点炫耀的意思。”

路易十四脚上还踩着一双红色的高跟鞋，好让他的个子能显得高一些。

“那他有多高啊？”

“1米63。”

“那也不矮啊。”

“对于男人来说偏矮了点。”

“对女的就算高的了。”

“还行。”

今天故事的红帽子指的是红衣主教，而红高跟鞋就是路易十四了。

路易十四在凡尔赛建了一座金碧辉煌的宫殿，那就是凡尔赛宫。凡尔赛宫的建造是路易十四集中政治权力的策略之一。路易十四將贵族们变成了他宮廷的成员，解除了他們作为地方長官的权利，藉此削弱了贵族的力量,為此他建造了凡尔赛宫。1682年5月6日他搬進這座位於巴黎城郊的巨大的宮殿。宮廷的規矩迫使贵族们為了衣裝費用而付出巨款，他們從早到晚都得待在宮殿裡參加舞會、宴席和其他庆祝活动。据说路易十四记忆惊人，当他进入大厅后一眼就可以看出谁在场，谁缺席,因此每个希望得宠于国王的贵族都必須每天在場。路易十四让這些贵族们沉溺於博取国王的宠幸，沒有时间去管理地方的问题，渐渐地他們就喪失統治地方的权利了。

那些有幸称为路易十四宠臣的人，日子自然过得非常舒适。但“羊毛都出在羊身上”，百姓们要缴纳大量的财务。这样路易十四才有钱来举办各式各样的聚会、舞会和宴会，才能送给朋友贵重的礼品。基于这样的社会状况，我们很快就会可拿到后来发生了什么。穷人们再也无法忍受这样的剥削和压迫，用一句俗话来说：“狗急了还会”

“爬树。”

“哈哈哈，是跳墙。”

“对对，狗急跳墙。”

“我们在以前讲过天翻地覆，法国大革命的故事。那里就说到路易十六和他的王后玛丽.安托瓦内特。他们都年少无知，玛丽.安托瓦内特在听到她的臣民们没有面包吃的时候，居然说’让他们吃蛋糕吧。’”
