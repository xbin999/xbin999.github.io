听取蛙声一片

有天在父与女上发了怪声，引起了好多朋友的兴趣。有“听取蛙声一片”，也有“伴着蛙声入眠”的。看来大家都挺向往田园生活。

小桥明月，稻田蛙声，该是多么有诗意的生活，可惜不是。不过虽然没有农村的风景，但我们有田园的兴致，自己找乐，寻找儿时的乡村野趣。晚上照例散步，靠近小区游泳池前的河道，就听到震天响的蛙声，“呱呱”不止，此起彼伏，遥相呼应。

近前就看到一只青蛙伏在水坑中彩色灯光上，黑黑灰灰的，青蛙蛤蟆也有点难分，不过小胳膊小腿倒甚是粗壮，小乐拿枝条一逗它，就一蹦，来个标准的蛙泳姿势，隐入黑暗中，我们也寻它不到。不过看它游地那么欢，倒也唤起了一家三口心中的游泳年头，夏天总算快到了。

听着青蛙欢快地歌声，把我们玩兴都勾起来了，乐妈回忆小时钓青蛙。小乐听听，就说：“我们要带手电筒来就好了。”

照例手机搞定一切，打开手电筒应用，倒也真是颇亮，于是一家三口就拿着手机电筒一只只找起青蛙来，找一只逗一只，小乐还老想拿树枝把青蛙捞起，可惜总落空。后来碰到一小的，估计也就比大蝌蚪大一点，被树枝从底往上一提，它就伏在枝条上，竟然真被捞出水面。

我说：“这青蛙开始飞行了，象伊索寓言中的老鹰带着飞行的乌龟。”（小乐最近听的英文故事）

不过看来这青蛙胆子没那么大，一动也不敢动，毫无自己飞行的想法，或者来个漂亮的跳水动作。小乐又小心翼翼地把它放下，一入水，倒又呈现了游泳健将的风范。

我又戏笑：“小乐，它是你的青蛙王子。”

“不要。”断然拒绝，头也不抬继续她的探索。

小区里也能有田园乐趣，只要去发现，看来下次散步我们得带手电筒了。

作文虚构和虚假的区别

前两天乐做一份语文的竞赛卷，里面的作文是写一个有趣的夏天的故事。小家伙很烦恼，夏天还没有到，去年的夏天又离的有点远，她都不太记得清发生过什么事了。我也觉得这个题目出的有问题，不上不下的，写春天也比这个强。
我想了会，觉得如果一定让乐乐写夏天的事，乐乐因为记忆模糊了，八成会编造，在我看来这样子写作文还不如让乐乐自己虚构一篇来的更健康。于是对乐乐提议“要不你写一篇想像作文吧，编一个童话故事，夏天发生的童话故事，这个题目是比较难写，我们就按自己的想像写好了，不要去管到时的分数高低！” 乐乐有点犹豫，后来也实在觉得想不出写什么，就同意了。
那天晚上刚好老姐约她去吃日本料理，所以写了一半，就出去吃饭了。等第二天LG看这篇作文时，竟然哈哈大笑起来。弄的在边上的乐乐弱弱地问：“怎么了，我写的不好吗？” 我也疑惑地看着LG，LG忙不迭地说：“没有没有，写的很好，写的太有意思了！” 我们才松了一口气，我接过手，果然，结尾太搞笑了。乐乐也开心地说：“我也觉得自己写的不错！”晚上，我还鼓励她可以一直写小狐狸的系列故事，可以给小狐狸取个像倒霉呀什么的名字！可小乐像保持自己小宝宝一样不忍心地说：“不行，我不想这样，就小狐狸的故事好了！”
然后就有了上篇文章里写的，乐乐因为考的不好，反而表扬我的事。这篇作文，满分30分，乐乐一下被扣了12分。小家伙有点难过说：“妈妈，如果作文能少扣点，我就分数很高了！” 我马上安慰她：“没事的，本来我们就不在乎分数，你写的真的很有趣！” 文化有限，所以不知道如何安慰她， 好在小乐也马上说：“我也觉得自己写的挺好的！妈妈，你帮我打印下来吧，把它夹在文件夹里！”
晚上和LG说了这事，LG让我告诉小乐，他把她写的作文放到网上，大家都说写的很好，如果她继续写下去，写到100篇时，他就给她出本书。我想这是我们家长的鼓励，但孩子光有家长的爱是不够的。她也需要老师和分数的认同，怎么才能让她理性的看待这些呢？
我想到的还是《好妈妈》，记得里面有关于写作文的文章。昨天就翻看了，我很庆幸自己当时做对了，让乐选择了“虚构” 而没有选“虚假”，“写作中的虚构与虚假是完全不同的两回事，这实质上是有想象力与缺乏想象力的区别。基于真情实感的虚构，是具有想象力的美的东西；虚假的文字是缺少真情实感和想象力的勉强之作，不会有美在其中。”
晚上给乐乐念故事就选了这篇。里面有三个孩子写作文的故事，其中一个女孩子帮妈妈以干家务，后来作文写“觉得应该好好学习，担心长大后找不到工作，就得去给别人当保姆。”我问乐乐，她写的好吗？乐乐说“恩，写的挺好的！”很高兴，乐乐还是纯真的乐乐 ，没有被套路搞晕。可是女孩子的老师却让也重写，因为认为对保姆不该瞧不起。说了真话的作文，让重写，是什么道理呀？ 最后一个是讲作者女儿的故事 ，这个故事对乐触动有点大。圆圆的作文是给妈妈洗脚，然后写一篇感受。圆圆在妈妈的话中，懂了作文本身含义，选择放弃洗脚，改成陪妈妈散步。乐乐紧张地问：“那她的作文怎么写？” 作者的话给了乐和我鼓励：“写作文时，尤其面对一个命题作文时，要调动自己的诚意。因为题目来自老师，乍一看题目，可能自己一下找不到感觉，不知该写什么，那么在动笔之前一定要问自己：就这个题目或这方面内容，我是如何理解的，我最想说什么，我有和别人不一样的想法吗，我最真实的想法到底是什么。” 最后圆圆就写了陪妈妈散步，我想这个对乐乐是有触动的。让她知道，当为了作文分数而去虚假时，就磨灭了人性，滑入功利平庸。所以我让乐乐不要在乎作文的分数，还表扬了她以前写过的两篇作文，因为我也在故事中， 所以我知道她写的真不真实。这两篇作文，虽然谈不上好，但是句子通畅，情感真切，最重要的是如实反映了当时的情形。
又是一条漫漫长路呀！

——————————————————

以上来自乐妈的文章。在面临我们的教育国情时，如何写真实的感受就更难了。老师引导的往往还是如何才能获得一个高分，就像上文中的例子所说，思想是不是健康向上，有没有意义之类的，而忽略了孩子的童真乐趣。而作为家长则需要顽强地保护孩子的一份纯真。

手工制作华容道

华容道于我是儿时的游戏，印象中老爸在我上小学的时候曾给我买过一个，记得最少需要72步才能走出。不过那时很少有走出，还是靠记步骤完成的。清明去西塘玩的时候在小店看到了，把这段记忆又给翻了出来，原想淘宝上买一个来着，转念一想又不难，何不自己做一个。

于是翻箱倒柜，找废纸板类的废物利用。材料如下：

泡沫垫子、鞋盒、尺子、铅笔、剪刀、胶带纸、颜色笔。

  • 在泡沫垫子上用尺子画好棋子大小，曹操（2x2），五虎上将（2x1），外加4个小兵（1x1），然后用剪刀剪下即可。
  • 将鞋盒四周边剪去，只余一点高度。再画好格子（4x5），比棋子略大点（主要棋子不是很平整），方便移动，边上用胶带胶一圈，使得硬朗些。
  • 接下来就是画像，选择了三国杀Q版的图形，少了个黄忠，另外找了个补数。
    全家齐上阵，我画画，乐妈描线，小乐负责填色、写字，再有乐妈完成塑封（用胶带）。

大功告成，看成果如何！反正我们自己都是赞不绝口，哈哈。最重要的还是一起玩的快乐时光。

夏天的故事

在一所动物学校里，同学们正在抢一个大西瓜。

这西瓜长得又大又圆，从那碧绿的外皮就能看出来。这么好的西瓜能不抢吗？在烈日下，所有动物们都摩拳擦掌，准备随时扑上前去把西瓜抱住。

可是，狐狸却没有跟其他一样。他只是用他的三寸不烂之舌对好多脾气暴燥、头脑简单的动物重复一句话：＂这个西瓜有毒，吃了要死的!＂而这些动物就傻乎乎地相信了。

对那些机灵的动物，狐狸另有办法对付。他先悄悄对聪明的动物们说：＂伙计们，我家附近有一块瓜田，结满了解渴的大西瓜，想去的跟我来！＂说着转身跑出学校，后面跟着一大帮小动物。

狐狸领着他们拐了几个弯，来到了一个小胡同里，狐狸对这里可是熟门熟路。他带着小动物们左弯右拐，很快就把他们丢在了后面。他得意扬扬地跑回学校，却发现西瓜已经被吃光了，原来西瓜被老师刮分了。

给小乐讲人类的故事之某某、某某人

在给小乐讲完《万历十五年》后，开始了《人类的故事》。人类的故事这几天断断续续地和小乐说了“埃及的故事”、“美索不达米亚”、“苏美尔人”、“摩西”、“腓尼基人”，每节都很短，但信息量其实非常大，尤其其中的苏美尔人、闪米特人、希伯来人、腓尼基人等等实在对我们来说非常的不熟悉，估计听过并不能留下什么印象。

故事基本还是围绕着古埃及和古巴比伦王国来说的，两大文明故国，两者前面各有其悠久的历史，但有互有穿插。这里关联的有两条线：

一是摩西带领希伯来人走出埃及。希伯来人是闪米特人的一支牧羊部落，尝试着在巴比伦王国找牧场，结果被王国军队赶跑，来到了埃及。他们与埃及人共同生活了五个世纪，在埃及被希克索斯人侵占时，他们为希克索斯人效力。后来底比斯人革命，赶跑了希克索斯人，埃及再获自由。犹太人就大难临头了。之后由摩西带领犹太人出走埃及，来到了巴勒斯坦。

二是亚述征服了整个西亚后，埃及也成为了帝国的一部分。公元前7世纪的时候，埃及获得过独立。可惜好景不长，先是波斯人，再是亚历山大大帝，之后又是罗马人，都建立了庞大的帝国，古埃及和古巴比伦是同病相怜，都毫无例外地成为了帝国中的一部分。

古代文明的内容非常的丰富，比如说古埃及的胡夫金字塔、古巴比伦王国的空中花园都是世界七大奇迹；还有我们历史课学的汉谟拉比法典；古埃及人的象形文字，苏美尔人的楔形文字，还有腓尼基人创造的字母是我们现代字母的来源。我还得想个办法把这些历史给串一下，和小乐再分享一下。

你若以为小孩小，你比小孩还要小

转昨天乐妈的文章：

今天LG晚回家，讲故事就由我来全权负责。问小乐想听＜＜狼王梦＞＞还是＜＜好妈妈胜过好老师＞＞?她选择后者，说前者想自己看。

第一次看它，是因为姐姐看我对乐的教育方法，无力说服我，就送了它给我。我从小不喜欢看书，对它的第一感觉就是：只不过是单纯的炒作，讲着不痛不痒大道理的赚钱工具罢了! 一直没理会。第二次看到它，是我被乐乐幼儿园评上＂好妈妈＂的称号而得的奖品。当时我很惭愧，我除了一直热心幼儿园活动外，在教育乐乐上真的称不上可以标榜的榜样。

一下得到同样的两本书，那天我就打开看了。然后，对阅读有障碍的我，尽然饶有兴致的看完了一遍，又看一遍。边看边反思边懊悔，有的事例仿佛都有我的影子，我仿佛将灵魂跳出身体，一点点地审视着自己的过去，看得又羞又愧。而后面就是奇迹出现的时刻___我真的变了!

今晚给乐念了＂不要捉弄孩子＂，里面讲到电视台对五胞胎提的问题后：“你们中随最爱告状”，“谁最爱打别人”，孩子很容易受误导，看别人指谁就指谁，而被指的孩子则无所适从。

乐不禁厉声道：＂他们干吗呀?!脑子有没有问题呀?!＂

很多时候，我觉得多数人是有是非观的，只是不敢说出自己的观点。像捉弄孩子的事，很多家长都亲眼看着自己的孩子被别人捉弄过，碍于面子没好意思说。我现在也让自己管好自己的言行，不去捉弄孩子。

书中有一首陶行知先生写的小诗，和乐乐念了两遍，朗朗上口：＂人人都说小孩小，小孩人小心不小，你若以为小孩小，你比小孩还要小＂
 

最近我和乐妈两个人都在分别给小乐念书，也挺杂的，有《万历十五年》、《在爱里相遇》、《人类的故事》、《撒哈拉的故事》、《好妈妈胜过好老师》。我负责历史类，乐妈负责情感类，前面两本都念完了，而象《撒哈拉的故事》念着念着，小乐就忍不住自己把它给看完了。

编写markdown文档直接发布到wordpress上



我现在用nvALT来书写markdown文档，有时想把写好的文章发布到自己的博客上，比较土的方式往往是先预览，然后打开博客编辑，复制粘帖。

想想是不是可以简化一下，能直接把写好的markdown文档发布到wordpress博客上。试了一下，分三步走：

1. 可以把nvALT的存储模式设置成普通文本文件（Plain Text Files ），这样写好的内容就会以单个文件形式保存在设置的目录中。
2. 然后使用flavor 讲文件转换成html文件，需要安装ruby 和json。
3. 发布到wordpress上，这个原来想偷点懒，直接基于wordpress提供的发布邮件账号，通过邮件发出即可，但后来发现直接基于命令行上的mail命令，不支持html文本。只好另外想办法，找个API来直接调用接口发布。wordpress支持xmlrpc，有个简单的例子Simple WordPress Posting from Ruby via XMLRPC，直接搬来使用即可。
4. 最后做的事情就是把flavor和上面的例子串接一下，代码放在github上。
这篇文档就是以这个方式完成发布的，测试的过程中碰到几个问题： 运行flavor时报错

~/.rvm/gems/ruby-1.9.3-p125/gems/json-1.7.5/lib/json/ext/parser.bundle: [BUG] Segmentation fault
ruby 1.8.7 (2012-02-08 patchlevel 358) [universal-darwin12.0]
</pre> 这个原因是flavor.rb指定了/usr/bin/ruby，和我的rvm环境不一致，修改ruby运行环境即可。 * 命令行下mail 运行时报错 <pre>sendmail: fatal: chdir /Library/Server/Mail/Data/spool: No such file or directory

这里需要设置mac下的mail环境，有篇详细的说明。
* 中文字符的问题

xmlrpc/create.rb:46:in `join’: incompatible character encodings: UTF–8 and ASCII–8BIT (Encoding::CompatibilityError)

这里需要讲flavor调用github解析返回的结果做强制编码html.force_encoding(“UTF–8”).encode(“UTF–8”)。

2013-04-27

评“说说我的“民办小学的艰难路”，结果让人很心酸……”

微博上看到一帖子“说说我的“民办小学的艰难路”，结果让人很心酸……”，看得很恼火，自以为正确的母亲在那里做着傻事，还一副神圣的教育的样子。

摘几段，原文链接。

今天上午刚知道了结果，没进！让我感觉这在意料之外，也在意料之中，很心酸，女儿得知后竟然哭的晕过去了，让这么小的孩子就经历这么大的挫折，非常不忍心……只能说这社会太现实，太让我失望了……

今天是值得纪念的日子，因为我们一家人等这一天已经等了三年了，为什么这么说，因为家所在的学区很差，所以，三年前我就决定让女儿考民办，为此三年前全家就为考民办小学做出了艰辛的努力，立志让女儿成为一名开朗，胆大，好学，向上的孩子。

。。。

在这里真心的感谢外公，每天吃完晚饭，在楼下院子稍稍活动下，六点开始就要进行辅导了，也很难为孩子，听到别的孩子还在楼下院子里玩的开心呢，而她只能上楼做题，很残忍也很无奈，因为我们也没什么关系，也拼不了爹，

。。。

但是今天考完后，我问了孩子考试的内容，觉得有点失望，考题很简单，我不知道凭这样的考题，凭这短短的二十几分钟，老师就能对一个孩子做出“优秀”或者“不优秀”的评价？难道真如社会上所说的，进这学校全都是靠关系的？

。。。

妈妈说，现在最重要的，是让女儿从这件事的阴影中尽快走出来，开开心心的。
最后的一句话说“最重要的，是让女儿从这件事的阴影中尽快走出来，开开心心的”，但通篇的文章看到的就是抱怨不公平，而造成女儿哭地晕过去的原因又究竟在哪里？这位妈妈却一点没有去分析，只是认为“社会太现实”，“进这学校全都是靠关系”。

首先出发点就非常功利，三年的准备就是为进民办，而且把功利的出发点和正常的教育“立志让女儿成为一名开朗，胆大，好学，向上的孩子”混淆在一起，可能猛一看觉得教育目标很好，但上民办和“开朗、胆大、好学、向上”有关系吗？由于功利的念头，使得对考试的内容失望，这反而正说明了自己的现实。你的心里是怎么样的，使得你看到的也是什么样的。

其次就是把原因都归结为外部因素，“学区很差”，“我们没关系”，“考题很简单”，“进这学校全都是靠关系的”，似乎一切的一切都是社会、环境造成的。其实换一个学区，你一样会冲进去，这个世界上还是没有关系的多，并不是所有人都可以拼爹的。你剥夺了孩子快乐的童年，而恰恰能给予孩子怎样的快乐是当爸当妈的比拼的，父母可以比陪孩子了多少时间，可以比给孩子讲了多少故事，有没有和孩子捉迷藏啊，可以比少给一点孩子压力。

记得小乐在陪同学练琴去考试的时候，就说：“你慢慢来。”是的，你急什么呢？是你在把你急迫的心态带给了孩子，影响到了孩子。龙应台在“孩子你慢慢来”中写到：
“我，坐在斜阳浅照的石阶上，望着这个眼睛清亮的小孩专心地做一件事；是的，我愿意等上一辈子的时间，让他从从容容地把这个蝴蝶结扎好，用他五岁的手指。”

父母在剥夺了孩子的快乐的同时，也失去了陪孩子玩耍的乐趣。收手吧，父母对孩子最大的教育是影响，你的心态直接造成了孩子心灵上的阴影，如果你希望让孩子尽快从阴影中走出来，最好的办法就是改变自己。

写下此文也是给自己，不要犯同样的错误。
